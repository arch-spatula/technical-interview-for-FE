# `__proto__`, `[[proto]]`, `prototype`에 대해 설명해주시기 바랍니다.

자바스크립트는 멀티패러다임언어이고 prototype기반 언어입니다. 모든 자바스크립트의 자료형은 Object 객체를 상속하면서 만들어졌습니다. 이런 이유로 Object의 정적메서드를 많이 사용하고 동적메서드는 거의 금지되어 있습니다.
